# Dynamics 365 Field Service Schedule Board PCF

A custom Power Apps Component Framework (PCF) control that mimics the native Dynamics 365 Field Service Schedule Board UI.

## Features

- ✅ **Hourly View** – Gantt chart with time slots (7:00 AM – 6:00 PM), bookings positioned by start/end time
- ✅ **Daily View** – Day columns showing all bookings for each day in the selected range
- ✅ **Weekly View** – Week columns with aggregated total hours per week
- ✅ **Monthly View** – Calendar-style month view with day summaries
- ✅ **Resource Panel** – Left panel with search, sort, and utilization bars
- ✅ **Open Requirements** – Bottom panel with table of unscheduled requirements
- ✅ **Zoom Controls** – Adjustable timeline zoom level
- ✅ **Date Navigation** – Chevron controls to shift date ranges

## Tech Stack

- React 17 + TypeScript
- React Icons
- CSS Grid / Flexbox
- Power Apps Component Framework (PCF)

## Usage

This control can be added to any Model-Driven App form, Canvas App, or Custom Page.
